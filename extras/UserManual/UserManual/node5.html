Running PLPTool

PLPTool requires a http://www.oracle.com/technetwork/java/javase/downloads/index.htmlJava Runtime Environment that complies with at least Java 2 Platform SE 5 (1.5) specifications. PLPTool is shipped with http://rxtx.qbang.org/wiki/index.php/Main_PageRXTX library for serial communication in Windows. If you use Linux, you will have to install the library provided by the distribution that you use. Serial communication is used to download programs to the board.

PLPTool is located in sw/PLPTool in the PLP download archive. You need to uncompress this folder to run the software. If you use Windows, you need to run the batch file PLPToolWin32.bat if you use the 32-bit version of Windows, or PLPToolWin64.bat if you use the 64-bit version. PLPTool will give a warning if it can not detect the RXTX library, and the user will not be able to program the board or use the serial terminal.
